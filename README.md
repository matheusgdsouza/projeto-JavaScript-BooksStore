# 📚 Books Store

Projeto desenvolvido com foco em praticar conceitos de **JavaScript**, manipulação de dados e construção de interfaces dinâmicas.

---

## 🚀 Sobre o projeto

O **Books Store** é uma aplicação web que simula uma loja de livros, permitindo visualizar, filtrar e organizar uma lista de livros de forma dinâmica.

O principal objetivo deste projeto foi aplicar conceitos importantes de JavaScript moderno, especialmente:

- Manipulação de arrays
- Renderização dinâmica no DOM
- Organização de código
- Boas práticas de desenvolvimento front-end

---

## 🧠 Funcionalidades

- 📖 Listagem de livros
- 🔎 Filtro por categoria
- 💰 Ordenação por preço
- 📦 Exibição de livros disponíveis
- 🔄 Atualização dinâmica da interface

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript (ES6+)
